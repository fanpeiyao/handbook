Tags Input

tagsinput 用来放在input输入框的小标签

基本形式

直接拷贝以下代码来实现

alpha beta gama 
						

<div class="fn-tagsinput">
   <span class="tag fn-badge fn-badge-primary">alpha
       <span data-role="remove"></span>
   </span>
   <span class="tag fn-badge fn-badge-primary">beta
      <span data-role="remove"></span>
   </span> 
   <span class="tag fn-badge fn-badge-primary">gama
      <span data-role="remove"></span>
   </span>
  <input type="text" placeholder="" style="width: 3em !important;">
</div>

使用方法

通过 Data API

通过 input 标签中添加 data-role="tagsinput" 来进行设置 

						
<input id = "tags-1" type="text" value = "alpha,beta,gama" data-fn-tagsinput >

通过 JS

组织好 HTML 以后，可以通过 Javascript 调用。

					
//获取
$('#tags-1').val();
//"alpha,beta,gama"

$('#tags-1').tagsinput("items");
//["alpha","beta","gama"]

//增删
$('#tags-1').tagsinput('add','sometag');
$('#tags-1').tagsinput('remove','sometag');
$('#tags-1').tagsinput('removeAll');
