Smooth Scroll

平滑滚动插件，不支持IE9以下浏览器。

使用演示

滚动到底部


<button id="doc-scroll-to-btm" class="fn-btn fn-btn-primary">滚动到底部</button>
<script>
		$("#doc-scroll-to-btm").on("click", function() {
			var o = $(window);
			o.smoothScroll({
				position : $(document).height() - o.height()
			})
		});
</script>

滚动到顶部


<button data-fn-smooth-scroll class="fn-btn fn-btn-success">滚动到顶部</button>

定义选项


<button data-fn-smooth-scroll="{position: 57, speed: 5000}"
		class="fn-btn fn-btn-danger">慢悠悠地滚到距离顶部 57px 的位置</button>

使用方法

通过 Data API

在元素上添加 data-fn-smooth-scroll 属性。 

						
<button data-fn-smooth-scroll class="fn-btn fn-btn-success">滚动到顶部</button>

如果要指定滚动的位置，可以给这个属性设一个值。

						
<button data-fn-smooth-scroll="{position: 189}" class="fn-btn fn-btn-secondary">...</button>

通过 Javascript

方法

为了保证不同浏览器兼容，请在 $(window) 上调用 $().smoothScroll() 方法。 

		
$(window).smoothScroll([options])

	
						
// 滚动到底部
$('#my-button').on('click', function() {
  var $w = $(window);
  $w.smoothScroll({position: $(document).height() - $w.height()});
});

选项说明

参数 类型 默认 描述
position 数值 0 要滚动到位置，默认为 `0`，即滚动到顶部
speed 数值 750 ~ 1500 滚动速度，单位 `ms`，默认为 `750 - 1500`，根据距离判断
