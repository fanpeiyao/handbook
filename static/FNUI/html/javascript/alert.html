Alert

显示可关闭的页内警告信息。

使用演示

基本形式

不同的一个警告框
		
<div class="fn-alert">不同的一个警告框</div>

关闭按钮

带关闭按钮的警告框

		
<div class="fn-alert" data-fn-alert>
	<button type="button" class="fn-close">×</button>
	<p>带关闭按钮的警告框</p>
</div>

不同状态

成功状态

警告状态

危险状态。

默认状态

		
<div class="fn-alert fn-alert-success" data-fn-alert>
	<button type="button" class="fn-close">×</button>
	<p>成功状态</p>
</div>

<div class="fn-alert fn-alert-warning" data-fn-alert>
	<button type="button" class="fn-close">×</button>
	<p>警告状态</p>
</div>

<div class="fn-alert fn-alert-danger" data-fn-alert>
	<button type="button" class="fn-close">×</button>
	<p>危险状态。</p>
</div>

<div class="fn-alert fn-alert-secondary" data-fn-alert>
	<button type="button" class="fn-close">×</button>
	<p>默认状态</p>
</div>

调用方式

通过 Data API

在 Alert 最外层元素上添加 data-fn-alert。 


<div class="fn-alert">
	...
</div>

通过 JS

开启关闭按钮交互功能：

	
$('.fn-alert').alert()

方法

  • $().alert() - 激活 Alert 元素关闭按钮的交互功能。
  • $().alert('close')：直接关闭元素。

自定义事件

事件名称 描述
close.alert close方法被调用时立即触发
closed.alert 元素被关闭以后触发（CSS 动画执行完成） 
		
$('#my-alert').on('closed.alert', function(){
  alert('警告窗口已经关闭');
});

JS 控制示例

示例1：激活 Alert 元素关闭按钮的交互功能

在应用 JS 激活之前这个警告框是无法关闭的。不信点右边的 x 试试。


$(function() {
	$("#doc-alert-btn-bind").one("click", function() {
		$("#your-alert").alert(), alert("已激活，点击上的 X 试试！")
	})
});

示例2：使用 JS 关闭警告框及监听自定义事件

这是一个警告框！


$(function() {
	$("#doc-alert-btn-close").one("click", function() {
		$("#your-alert-1").alert("close")
	}), $(document).on("close.alert", "#your-alert-1", function() {
		alert("警告框开始关闭！")
	}), $(document).on("closed.alert", "#your-alert-1",
			function() {
				alert("警告框关闭完成！")
			})
});
