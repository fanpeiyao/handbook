OffCanvas

侧滑插件。

使用示例

该组件由触发器和侧滑元素两部分组成。触发器上添加 data-fn-offcanvas 属性，侧滑元素使用固定的 HTML 结构。

默认样式

点击显示侧边栏
  • 一个简单的列表
  • 通过样式美化
  • 就是一个正常侧边栏

<a href="#doc-oc-demo1" class="fn-btn fn-btn-primary" data-fn-offcanvas>点击显示侧边栏</a>
<div id="doc-oc-demo1" class="fn-offcanvas">
	<div class="fn-offcanvas-bar">
		<div class="fn-offcanvas-content">
			<ul>
				<li>一个简单的列表</li>
				<li>通过样式美化</li>
				<li>就是一个正常侧边栏</li>
			</ul>
		</div>
	</div>
</div>

Push 效果

设置 effect: 'push' 。

  • 一个简单的列表
  • 通过样式美化
  • 就是一个正常侧边栏

data-fn-offcanvas="{target: '#doc-oc-demo2', effect: 'push'}"

右侧显示

边栏默认显示在左侧，在内容容器上添加 .fn-offcanvas-bar-flip class 调整为右侧。

  • 一个简单的列表
  • 通过样式美化
  • 就是一个正常侧边栏

网易音乐 


<div id="doc-oc-demo3" class="fn-offcanvas">
	<div class="fn-offcanvas-bar fn-offcanvas-bar-flip">
		<div class="fn-offcanvas-content">
			...
		</div>
	</div>
</div>

调用方式

首先，按照以下结构组织好侧栏内容：

						
<div id ="your-id"  class ="fn-offcanvas" >
<div class="fn-offcanvas-bar">
    <!-- 你的内容 -->
  </div>
</div>

通过 Data API

在要触发侧栏的元素上添加 data-fn-offcanvas 属性：

  • 如果是 <a> 元素，则把 href 的值设置为边栏的 ID：href="#your-id"；
  • 如果是其他元素，则在 data-fn-offcanvas 的值里面指定侧边栏 ID： 

<button data-fn-offcanvas="{target: '#your-id'}"></button>
属性 描述
{target: '#your-id'} 指定侧边栏，如果是 a 元素则写在 href 里
{effect: 'push'} 边栏动画效果，可选的值为 overlay | push 默认为 overlay

通过 JS

组织好 OffCanvas HTML 以后，可以通过 Javascript 调用。

						
$('#my-offcanvas').offCanvas(options);

注意： 这里 #my-offcanvas 直接指向侧边栏元素，而不是触发器。

参数说明

  • options.effect，值为 overlay | push，默认为 overlay。
						
$('#my-offcanvas').offCanvas({effect:'push'});

通过 $().offCanvas(options) 设置，参数同上。

方法

  • $().offCanvas(options) - 设置边栏参数并打开边栏
  • $().offCanvas('open') - 打开边栏
  • $().offCanvas('close') - 关闭边栏

事件

事件名称 描述
open.offcanvas 打开边栏时立即触发
close.offcanvas 关闭边栏时立即触发

JS 控制示例

下面的示例演示了使用 JS 打开/关闭侧边栏。侧边栏打开以后，关闭按钮无法点击到，可以在控制台输入以下代码模拟点击事件：


$('[data-rel="close"]').click();
