使用方式

通过 Data API

在 radio / checkbox 上添加 data-fn-ucheck 属性。 

		
<label class="fn-checkbox">
  <input type="checkbox" value="" data-fn-ucheck> 没有选中
</label>

<label class="fn-radio">
  <input type="radio" value="" data-fn-ucheck> 没有选中
</label>

通过 JS

		
$(function() {
  $('input[type='checkbox'], input[type='radio']').uCheck();
});

方法

  • $().uCheck('check'): 选中
  • $().uCheck('uncheck'): 取消选中
  • $().uCheck('toggle'): 切换选中状态
  • $().uCheck('disable'): 禁用
  • $().uCheck('enable'): 启动
