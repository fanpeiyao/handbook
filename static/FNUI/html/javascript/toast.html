Toast

显示一个延时关闭的对话框。

使用演示

基本形式

fnui提供两种形式的toast弹出提示框，应该有三种提示，分别为信息、警告和加载中。通过FNUI.toast.show(type,time) 方法来显示toast

type为类型，接受'info','warning','loading'， 第二个参数time为定时关闭时间，如果不设定，则需要手动关闭

方法

  • $.toast.show('info',2000); - 提示信息；2s后自动关闭
  • $.toast.show('warning',3000); - 提示警告信息；3s后自动关闭
  • $.toast.show('loading',2000); - 加载信息；2s后自动关闭
  • $.toast.close() - 手动关闭

$.toast也等同于FNUI.toast

		
$.toast.show('warning');
$.toast.show('info');
$.toast.show('loading');

FNUI.toast.show('warning');
FNUI.toast.show('info');
FNUI.toast.show('loading');
//关闭
var toast = $.toast.show();
...
...
...
toast.close();

自定义内容

通过content()方法来自定义内容;通过clear()来清除自定义内容

调用过一遍content()之后，如果不进行清除clear()或者覆盖content()，自定义提示在别的地方调用不会删除。

  • $.toast.content()-自定义内容
  • $.toast.clear()-清除自定义内容
		
$('#toastpop3').click(function () {
	$.toast.content("自定义提示").show('info',2000).clear();
});

//或者
var toast = $.toast.content("自定义提示").show('info');
....
...
...
toast.clear();
toast.close();
