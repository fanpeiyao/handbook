Switch

switch 一个类似开关的checkbox按钮

基本形式


<input id="" type="checkbox" data-fn-switch>

颜色设定

通过属性 data-on-color 来设置打开按钮颜色， 通过 data-off-color 来设置关闭按钮颜色， 可以设定的颜色有 default、secondary、success、warning、primary 


<input type="checkbox" data-fn-switch data-on-color="success">
<input type="checkbox" data-fn-switch data-off-color="warning">

文字设定

通过属性 data-on-text 来设置打开按钮文字， 通过 data-off-text 来设置关闭按钮文字， 


<input type="checkbox" data-fn-switch data-on-text="打开打开"> 
<input type="checkbox" data-fn-switch data-off-text="关闭关闭">

使用方法

通过 Data API

通过input标签中添加data-fn-switch来进行设置 

	 
<input id="" type="checkbox" checked="checked" data-fn-switch disabled>
