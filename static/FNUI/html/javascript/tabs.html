禁用触控操作

在过长的 Tabs 中滚动页面时会意外触发 Tab 切换事件，用户可以选择禁用触控操作。

		
<div class = "fn-tabs" data-fn-tabs = "{noSwipe: 1}" >
  	    。。。。
</div>

调用方式

通过 Data API

在选项卡容器 .fn-tabs 上添加 data-fn-tabs 属性。上面的演示即通过此种方式调用。

通过 JS

方法

通过 $().tabs(options) 开启选项卡的交互功能。

  • 彩虹
  • 画面
  • 窗外

$(function() {
	var n = $("#doc-my-tabs");
	n.tabs(), n.find("a").on("opened.tabs", function() {
		console.log("[%s] 选项卡打开了", $(this).text())
	})
});

选项

  • options.noSwipe 是否禁用触控事件。

$('#someTabs').tabs({noSwipe: 1});

事件

事件触发在标签上。

事件名称 描述
open.tabs 打开一个选项卡时立即触发
opened.tabs 选项卡打开完成时触发（CSS 动画执行完成） 

$(function() {
	var n = $("#doc-my-tabs");
	n.tabs(), n.find("a").on("opened.tabs", function() {
		console.log("[%s] 选项卡打开了", $(this).text())
	})
});
