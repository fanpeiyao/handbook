Button JS 交互

Button 及 Button group 与 JS 交互。

按钮 loading 状态

默认文字

默认的文字为 loading...。

  

<button type="button" class="fn-btn fn-btn-primary btn-loading-example">按钮- button 元素</button>
<input type="button" class="fn-btn fn-btn-secondary btn-loading-example" value="按钮 - input 元素" />

	
		
$(function() {
	$(".btn-loading-example").click(function() {
		var t = $(this);
		t.button("loading"), setTimeout(function() {
			t.button("reset")
		}, 5e3)
	})
});

自定义选项

可以在元素上添加 data-fn-loading 来设置选项：

  • spinner 加载动画图标，适用于支持 CSS3 动画、非 input 元素，写图标名称即可；
  • loadingText 加载时显示的文字， 默认为 loading；
  • resetText 重置以后的显示的文字，默认为原来的内容。
  

<button type="button" class="fn-btn fn-btn-primary btn-loading-example" data-fn-loading="{spinner: 'circle-o-notch', loadingText: '加载中...', resetText: '加载过了'}">按钮- button 元素</button>

<input type="button" class="fn-btn fn-btn-secondary btn-loading-example"
	value="按钮 - input 元素" data-fn-loading="{loadingText: '努力加载中...'}" />

单按钮状态切换

按钮状态：未激活 


$(function() {
	function t() {
		var t = n.hasClass("fn-active") ? "未激活" : "激活";
		$("#doc-single-toggle-status").text(t)
	}
	var n = $("#doc-single-toggle");
	n.on("click", function() {
		t()
	})
});

复选框

注意：由于 FastClick 的原因，触屏设备上使用时需要在 input 上添加 .needsclick ，否则无法获取复选框的值。 


$(function() {
	var n = $('[name="doc-js-btn"]');
	n.on("change", function() {
		var c = [];
		n.filter(":checked").each(function() {
			c.push(this.value)
		}), console.log("复选框选中的是：", c.join(" | "))
	})
});
