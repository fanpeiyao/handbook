iScroll Lite

打包了 iScrolllite（当前版本为 5.1.3），参考官网。

iScroll 主要用来实现平滑的垂直、水平的拖动操作及控制。

通过 $.AMUI.iScroll 访问 IScroll 对象。 

						
<div id="wrapper">
  <ul>
    <li>...</li>
    <li>...</li>
    ...
  </ul>
</div>

					
						
var IScroll = $.AMUI.iScroll;
var myScroll = new IScroll('#wrapper');

常见问题

使用 iScroll 以后链接无法点击？

初始化 iScroll 的时候加上 click: true 参数即可 

						
var myScroll = new IScroll('#wrapper', {
  click: true
});
