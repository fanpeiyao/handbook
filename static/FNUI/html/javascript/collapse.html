Collapse

折叠效果组件，用于制作下滑菜单或手风琴效果。

不要在折叠内容的容器上设置垂直的 margin/padding/border样式。

使用演示

折叠面板

结合 Panel组件实现手风琴效果。需结合以下辅助 class 使用：

  • 要隐藏的内容添加 .fn-collapse；
  • 默认显示的内容添加 .fn-collapse.fn-in；

添加以上 class 以后，通过 Data API 来调用：

<h4 data-fn-collapse = "{parent: '#accordion', target: '#do-not-say-1'}"></h4>

其中：

  • parent 为容器 ID
  • target 为要伸缩的容器 ID

如果触发元素为 <a> 元素，可以把 target 设置在 href 属性里。 

						
<a data-fn-collapse="{parent: '#accordion'}" href="#do-not-say-1">
  ...
</a>

调用方式

通过 Data API

在元素上添加 data-fn-collapse 并设置 target 的值为折叠元素 ID： 


<button data-fn-collapse="{target: '#my-collapse'}"></button>

通过 JS

使用方法：


$('#myCollapse').collapse()

方法

  • $().collapse(options) - 绑定元素展开/折叠操作

$('#myCollapse').collapse({
  toggle: false
})
  • $().collapse('toggle') - 切换面板状态
  • $().collapse('open') - 展开面板
  • $().collapse('close') - 关闭面板

选项

参数 类型 默认 描述
parent 选择符 false 如果设置了 parent 参数，且该容器下有多个可折叠的面板时，展开一个面板会关闭其它展开的面板。换言之，如果想让多个面板可以都处于展开状态，那不设置这个参数即可。
toggle 布尔值 true 交替执行展开/关闭操作

事件

事件在折叠的元素上触发，以上面的折叠菜单为例，#collapse-nav 触发事件： 


	$(function() {
		$("#collapse-nav").on("open.collapse", function() {
			console.log("折叠菜单打开了！")
		}).on("close.collapse", function() {
			console.log("折叠菜单关闭！")
		})
	});
事件 描述
open.collapse open 方法被调用时立即触发
opened.collapse 元素完全展开后触发
close.collapse close 方法被调用后立即触发
closed.collapse 元素折叠完成后触发
