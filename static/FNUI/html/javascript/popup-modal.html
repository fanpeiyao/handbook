Popup-Modal

Popup-Modal 交互窗口，经过封装的modal窗口，可以用来表达 alert 、 confirm 、 prompt 窗口。

使用演示

使用过程中需按示例代码组织好 HTML。

基本形式

此 Demo 设置了 closeViaDimmer 、 width 、 height 参数。 

/*						
 * options:{
 *      type: popup(default), alert, prompt, loading, action(待开发)
 *      title:   title 
 *      content:  body content  string or html
 *      oncancel:    only prompt
 *      onconfirm:   only prompt
 *      }
 */
 
						$("#default-popupmodal").on("click",function(){
						
						$.modal(
							{title:"这是一个测试窗口",content:"这是测试窗口弹出来的内容内容内容内容内容"}
						);
						
						});

参数options

框架根据调用的参数options的不同，可以展示不同的内容

						
							$.modal(
							{
							type:"alert",
							title:"警告",
							content:"内容，需要警告的事情"
							}
							
							);
							$.modal(
							
							{
								type:"prompt",
								title:"你确定这么做嘛?",
								content:"如果删除，将不可恢复",
								oncancel:cancel,
								onconfirm:confirm
							}
							
							);
							$.modal(
							
							{type:"loading"}
							
							);
