Dialog

使用演示

创建一个对话框，不加遮罩

运 行 
						
					
$.dialog({
				    content : '对话框内容',
				    title : 'ok',
				    ok : function() {
				        alert('我是确定按钮，回调函数返回false时不会关闭对话框。');
				        return false;
				    },
				    cancel : function() {
				        alert('我是取消按钮');
				    },
				    lock : false
});

2秒自动关闭

运 行 
						
 $.dialog({
             content : '窗口将在2秒后自动关闭',
             title: "alert",
              width: 600,
             time : 2000
  });

加载中，不会消失

运 行 
						
 $.dialog();
