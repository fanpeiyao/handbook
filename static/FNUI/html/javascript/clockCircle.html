ClockCircle

圆形进度条，基于 CSS3 transition 实现圆形进度条；CSS3 动画实现进度操作。

使用演示

设定默认进度条

70
						
<div  data-fn-clockcircle> 70</div>

调用方式

通过 Data API

在选项卡容器上添加 data-fn-clockcircle 属性。

40
						
<div  data-fn-clockcircle> 40</div>

通过 JS

方法

通过 $("#id").clockCircle(options) 来调用。 

						
<div class="fn-tabs" id="doc-my-clockcircle"></div>
<script>
/**
 *
 *     	创建：$("div").clockCircle(options);
 *            options = {
 *                  size: "60px"     //大小
 *                  width: "5px"     //环的粗细
 *                  color: "#0ac"    //环的颜色
 *                  percent:40       //当前百分比
 *            }
 *      方法： $("div").rotate(percent)    percent:数字    //转过的百分比
 *      方法： $("div").process(percent,speed)  speed:"fast" "median" "slow"  //默认fast 500ms
 *
 * 		支持链式调用 $("div").clockCircle().rotate(30).process(50);
 * 		不支持多个process 连续调用
 *
**/
  $(function() {
    $('#doc-my-clockcircle').clockCircle({percent:10,size:100,width:"2px"});
  })
</script>

选项

  • options.size 环的大小
  • options.width环的粗细
  • options.color环的颜色(含中心数字)
  • options.percent进度百分比（数字，不带百分号）
						
$('#someclock').clockCircle({percent:10,size:100,width:"2px",color:"red"});

动画

通过动画可以设定进度条的进度，包含两个动画方法，可通过js来调用动画

  • $().rotate(percent) 改变百分比，无渐变
  • $().process(percent,speed)改变百分比，有渐变，可通过speed参数调节
						
$('#doc-my-clockcircle1')
.on('click',function(){$("#doc-my-clockcircle1").rotate(30);})
$('#doc-my-clockcircle1')
.on('click',function(){$("#doc-my-clockcircle1").process(70,"slow");
				})
						//链式调用加载到100%
$('#doc-my-clockcircle2')
.on('click',function(){$("#doc-my-clockcircle2").clockCircle({size:"100px",width:"2px",color:"blue"}).process(100,"fast");
				})
