宽度适应

下拉内容 .fn-dropdown-content 为绝对定位，宽度会根据内容缩放（最小为 160px ）。

在 data-fn-dropdown 里指定要适应到的元素如{justify: '#doc-dropdown-justify'}，下拉内容的宽度会设置为该元素的宽度。

标题

内容


<div id="doc-dropdown-justify">
	<div class="fn-dropdown"
		data-fn-dropdown="{justify: '#doc-dropdown-justify'}">
		<button class="fn-btn fn-btn-success fn-dropdown-toggle">
			宽度适应下拉 <span class="fn-icon-caret-down"></span>
		</button>
		<div class="fn-dropdown-content">
			<h2>标题</h2>
			<p>内容</p>
		</div>
	</div>
</div>

调用方式

通过 Data API

如上面的演示所示，根据示例组织好 HTML 结构，然后在 .fn-dropdown 上添加 data-fn-dropdown 属性，相关选项可以设置在该属性的值里。

通过 JS

按照示例组织好 HTML 结构（不加 data-fn-dropdown 属性），然后通过 JS 来调用。

标题

内容


$(function() {
	$("#doc-dropdown-js").dropdown({
		justify : "#doc-dropdown-justify-js"
	})
});

方法

  • $(element).dropdown(options) 激活下拉功能；
  • $(element).dropdown('toggle') 下拉状态交替；
  • $(element).dropdown('close') 隐藏下拉菜单；
  • $(element).dropdown('open') 显示下拉菜单。

$("#doc-dropdown-js").dorpdown("toggle");
$("#doc-dropdown-js").dorpdown("open");
$("#doc-dropdown-js").dorpdown("close");

参数说明

参数 类型 默认 描述
boundary 选择器 window 下拉内容边界，避免下拉内容超过边界被遮盖截断
justify 选择器 undefined 下拉内容适应宽度的元素

事件

下拉框的事件在.fn-dropdown上触发。

事件名称 描述
open.dropdown 调用显示下拉框方法时立即触发
opened.dropdown 下拉框显示完成时触发
close.dropdown 调用隐藏方法时触发
closed.dropdown 下拉框关闭完成时触发 

$(function(){
  $dropdown.on('open.dropdown', function(e){
    console.log('open event triggered');
  });
});
