Breadcrumb

.fn-breadcrumb 面包屑导航。

基本形式

默认分隔符

  1. 首页
  2. 分类
  3. 内容

<ol class="fn-breadcrumb">
	<li><a href="#">首页</a></li>
	<li><a href="#">分类</a></li>
	<li class="fn-active">内容</li>
</ol>

斜杆分隔符

增加一个fn-breadcrumb-slashClass

  1. 首页
  2. 分类
  3. 内容

<ol class="fn-breadcrumb fn-breadcrumb-slash">
	...
</ol>

结合 Icon

添加图标相应 class 即可。

  1. 首页
  2. 分类
  3. 内容

<ol class="fn-breadcrumb">
	<li><a href="#" class="fn-icon-home">首页</a></li>
	<li><a href="#">分类</a></li>
	<li class="fn-active">内容</li>
</ol>
