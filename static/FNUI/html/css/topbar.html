顶部、底部固定

顶部固定

在 .fn-topbar 上添加 .fn-topbar-fixed-top class，实现顶部固定。

包含顶部固定导航条的页面会在 body 上添加 .fn-with-topbar-fixed-top ，用户可以在这个 class 下设置样式，默认设置了 padding-top: 51px 。 

						
<header class="fn-topbar fn-topbar-inverse fn-topbar-fixed-top">
  <div class="fn-container">
...
  </div>
</header>

底部固定

在 .fn-topbar 上添加 .fn-topbar-fixed-bottom class，实现底部固定。

包含底部固定导航条的页面会在 body 上添加 .fn-with-topbar-fixed-bottom ，用户可以在这个 class 下设置样式，默认设置了 padding-bottom: 51px 。 

						
<header class="fn-topbar fn-topbar-inverse fn-topbar-fixed-bottom">
  <div class="fn-container">
...
  </div>
</header>
