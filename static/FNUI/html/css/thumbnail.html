Thumbnail

Thumbnail 组件主要用于显示图片列表及图文混排列表。

基本样式

在 <img> 添加 .fn-thumbnail 类；也可以在 <img> 外面添加一个容器，如 <div> 、 <figure> 、 <a> 等，再将 class 添加到容器上。

结合网格使用 

						
<div class="fn-g">
  <div class="fn-u-sm-4">
    <img class="fn-thumbnail" src="..." alt=""/>
  </div>

  <div class="fn-u-sm-4">
    <a href="#" class="fn-thumbnail">
      <img src="..." alt=""/>
    </a>
  </div>

  <div class="fn-u-sm-4">
    <figure class="fn-thumbnail">
      <img src="..." alt=""/>
    </figure>
  </div>
</div>

结合 AVG Grid 使用

添加 AVG Grid class 同时添加 .fn-thumbnails 。 

						
<ul class="fn-avg-sm-3 fn-thumbnails">
  <li>
    <img class="fn-thumbnail" src="..." alt=""/>
  </li>

  <li>
    <a href="#" class="fn-thumbnail">
      <img src="..." alt=""/>
    </a>
  </li>

  <li>
    <figure class="fn-thumbnail">
      <img src="..." alt=""/>
    </figure>
  </li>
</ul>

标题

图片标题 #1

						
<div class="fn-g">
  <div class="fn-u-sm-4">
    <div class="fn-thumbnail">
      <img src="..." alt=""/>
      <h3 class="fn-thumbnail-caption">图片标题 #1</h3>
    </div>
  </div>

  <div class="fn-u-sm-4">
    <a href="#" class="fn-thumbnail">
      <img src="..." alt=""/>
      <figcaption class="fn-thumbnail-caption">图片标题 #2</figcaption>
    </a>
  </div>
  <div class="fn-u-sm-4">
    <figure class="fn-thumbnail">
      <img src="..." alt=""/>
      <figcaption class="fn-thumbnail-caption">图片标题 #3</figcaption>
    </figure>
  </div>
</div>

图文混排

在 fn-thumbnail 内加入 .fn-caption 可以添加任何类型的HTML内容标题、段落、或按钮。

百年孤独选

连那最坚韧而又狂乱的爱情归根结底也不过是一种转瞬即逝的现实。
