Icon

使用 Font Awesome图标库，进行了简单封装

使用方法

添加 Class

在 HTML 上添加添加 fn-icon-{图标名称} class。

QQ Wechat 

<span class="fn-icon-qq">QQ</span> <span class="fn-icon-weixin">Wechat</span>

图标大小

  • .fn-font-sm，缩小 80%
  • .fn-font-md，放大 200%
  • .fn-font-lg，放大 250%

<i class="fn-font-audit fn-font-sm"></i> 
<i class="fn-font-audit fn-font-md"></i> 
<i class="fn-font-audit fn-font-lg"></i>

Icon button

在 Icon 上添加 .fn-btn-icon class。

如果需要设置 Icon Button 的颜色，可以添加以下 class：

  • .fn-primary
  • .fn-secondary
  • .fn-success
  • .fn-warning
  • .fn-danger

<a href="" class="fn-icon-btn fn-icon-twitter"></a>
<a href="" class="fn-icon-btn fn-icon-facebook"></a> 
<a href="" class="fn-icon-btn fn-icon-github"></a> 
<a href="" class="fn-icon-btn fn-primary fn-icon-qq"></a> 
<a href="" class="fn-icon-btn fn-secondary fn-icon-drupal"></a> 
<a href="" class="fn-icon-btn fn-success fn-icon-shield"></a> 
<a href="" class="fn-icon-btn fn-warning fn-icon-warning"></a> 
<a href="" class="fn-icon-btn fn-danger fn-icon-youtube"></a>

旋转动画

注意：Chrome 和 Firefox 下， display: inline-block; 或 display: block; 的元素才会应用旋转动画。 


<i class="fn-icon-spinner fn-icon-spin"></i> 
<i class="fn-icon-refresh fn-icon-spin"></i> 
<i class="fn-icon-circle-o-notch fn-icon-spin"></i> 
<i class="fn-icon-cog fn-icon-spin"></i> 
<i class="fn-icon-gear fn-icon-spin"></i>
<i class="fn-icon-spinner fn-icon-pulse"></i>

固定宽度

FontAwesome 在绘制图标的时候不同图标宽度有差异， 添加 .fn-icon-fw 将图标设置为固定的宽度，解决宽度不一致问题
