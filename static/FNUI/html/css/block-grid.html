AVG-Grid

Average Grid，均分网格（原 Block Grid），使用 ul/ol创建等分列，用于内容的排列。

响应式断点为：

Class 区间
.fn-avg-sm-* 0 - 767px
.fn-avg-md-* 768px - 1199px
.fn-avg-lg-* 1200px +

与布局网格不同的是，这里的数字表示几等分，比如 .fn-avg-sm-2 会将子元素 <li> 的宽度设置为 50% 。

考虑到通用性（菜单、图片）等， <li> 没有设置 padding ，使用时需根据需求自行设置。

另外需要注意的 AVG Grid 只能用于 ul/ol 结构 。

基本使用

只添加 .fn-avg-sm-* ，应用于所有屏幕尺寸。 


<ul class="fn-avg-sm-4 fn-thumbnails">
  <li><img class="fn-thumbnail" src="img/demos/01.jpg" /></li>
  <li><img class="fn-thumbnail" src="img/demos/02.jpg" /></li>
  <li><img class="fn-thumbnail" src="img/demos/03.jpg" /></li>
  <li><img class="fn-thumbnail" src="img/demos/04.jpg" /></li>
</ul>

响应式

按需增加更多响应式 class，缩放窗口可以查看响应效果。
