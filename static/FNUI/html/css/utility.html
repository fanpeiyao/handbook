响应式辅助

视口大小

.fn-[show|hide]-[sm|md|lg][-up|-down|-only] ，调整浏览器窗口大小查看元素的显隐。

class 释义：

  • show 显示，hide 隐藏，这应该不难理解；
  • sm、md、lg 是划分屏幕大小区间的缩写，对应 small、medium、large，网格里面做了说明；
  • down 指小于区间，up 指大于区间， only 指仅在这个区间。

按照上面的翻译一下下面的 class:

  • .fn-show-sm-only: 只在 sm 区间显示
  • .fn-show-sm-up: 大于 sm 区间时显示
  • .fn-show-sm: 在 sm 区间显示，如果没有设置 md、lg 区间的显隐，则元素在所有区间都显示
  • .fn-show-md-down: 小于 md 区间时显示
