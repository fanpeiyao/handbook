Close

关闭按钮样式，可以结合其他不同组件使用。对 <a> 或者 <button> 添加 .fn-close class。

基本样式

在元素上添加 .fn-close class。

×
						
<a href="#" class="fn-close">&times;</a>

<button type="button" class="fn-close">&times;</button>

带边框样式

添加 .fn-close-alt class。

使用 x

× 
						
<a href="" class="fn-close fn-close-alt">&times;</a>

使用 Icon Font 

						
<a href="" class="fn-close fn-close-alt fn-icon-times"></a>

hover 旋转

添加 .fn-close-spin class（需支持 CSS3 transform）。

× 
						
<a href="" class="fn-close fn-close-alt fn-close-spin">&times;</a>
<a href="" class="fn-close fn-close-alt fn-close-spin fn-icon-times"></a>
