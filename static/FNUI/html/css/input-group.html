Input Group

Input group 基于 Form 组件和 Button 组件扩展，依赖这两个组件。

在容器上添加 .fn-input-group ，在标签文字上添加 .fn-input-group-label ，具体请查看示例代码。

基本使用

输入框与标签

下面的代码中演示了结合 Icon 组件及添加文字的样式。

.00
$ .00 
						
<div class="fn-input-group">
  <span class="fn-input-group-label"><i class="fn-icon-user fn-icon-fw"></i></span>
  <input type="text" class="fn-form-field" placeholder="Username">
</div>

<div class="fn-input-group">
  <span class="fn-input-group-label"><i class="fn-icon-lock fn-icon-fw"></i></span>
  <input type="text" class="fn-form-field" placeholder="Password">
</div>

<div class="fn-input-group">
  <input type="text" class="fn-form-field">
  <span class="fn-input-group-label">.00</span>
</div>

<div class="fn-input-group">
  <span class="fn-input-group-label">$</span>
  <input type="text" class="fn-form-field">
  <span class="fn-input-group-label">.00</span>
</div>

圆角输入框与标签

下面的代码中演示了结合 Icon 组件及添加文字的样式。

 
<div class="fn-form-group fn-form-icon">
<i class="fn-icon-pencil fn-color-blue"></i>

</div>

复选/单选框与输入框

将单选框与复选框放入 .fn-input-group-label 内。 

						
<div class="fn-g">
  <div class="fn-u-lg-6">
    <div class="fn-input-group">
      <span class="fn-input-group-label">
        <input type="checkbox">
      </span>
      <input type="text" class="fn-form-field">
    </div>
  </div>
  <div class="fn-u-lg-6">
    <div class="fn-input-group">
      <span class="fn-input-group-label">
        <input type="radio">
      </span>
      <input type="text" class="fn-form-field">
    </div>
  </div>
</div>

输入框结合 Button

需要用 .fn-input-group-btn 包住按钮，而不是 .fn-input-group-label 。 

						
<div class="fn-g">
  <div class="fn-u-lg-6">
    <div class="fn-input-group">
      <span class="fn-input-group-btn">
        <button class="fn-btn fn-btn-default" type="button"><span class="fn-icon-search"></span> </button>
      </span>
      <input type="text" class="fn-form-field">
    </div>
  </div>
  <div class="fn-u-lg-6">
    <div class="fn-input-group">
      <input type="text" class="fn-form-field">
      <span class="fn-input-group-btn">
        <button class="fn-btn fn-btn-default" type="button">手气还行</button>
      </span>
    </div>
  </div>
</div>

样式变换

尺寸

在 .fn-input-group 添加标明尺寸的 class 即可。

包含 .fn-input-group-lg 、 .fn-input-group-sm 。

@
@
@ 
						
<div class="fn-input-group fn-input-group-lg">
  <span class="fn-input-group-label">@</span>
  <input type="text" class="fn-form-field" placeholder="Username">
</div>

<div class="fn-input-group">
  <span class="fn-input-group-label">@</span>
  <input type="text" class="fn-form-field" placeholder="Username">
</div>

<div class="fn-input-group fn-input-group-sm">
  <span class="fn-input-group-label">@</span>
  <input type="text" class="fn-form-field" placeholder="Username">
</div>
