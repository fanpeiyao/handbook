Button-group

组合 Button 元素。

基本使用

把一系列要使用的 .fn-btn 按钮放入 .fn-btn-group 。

 
  

<div class="fn-btn-group">
 <button type="button" class="fn-btn fn-btn-default">左手</button>
  ...
</div>

<div class="fn-btn-group">
  <button type="button" class="fn-btn fn-btn-primary fn-radius">左手</button>
  ...
</div>

<div class="fn-btn-group">
  <button type="button" class="fn-btn fn-btn-primary fn-round">左手</button>
  ...
</div>

按钮工具栏

将 .fn-btn-group 放进 .fn-btn-toolbar ，实现工具栏效果。 

<div class="fn-btn-toolbar">
  <div class="fn-btn-group">...</div>
  <div class="fn-btn-group">...</div>
  <div class="fn-btn-group">...</div>
</div>

按钮组大小

给 .fn-btn-group 增加 class .fn-btn-group-lg 或 .fn-btn-group-sm 或 .fn-btn-group-xs 改变按钮大小。 




						
<div class="fn-btn-toolbar">
    <div class="fn-btn-group fn-btn-group-lg">...</div>
</div>

<div class="fn-btn-toolbar">
    <div class="fn-btn-group">...</div>
</div>

<div class="fn-btn-toolbar">
    <div class="fn-btn-group fn-btn-group-sm">...</div>
</div>

<div class="fn-btn-toolbar">
    <div class="fn-btn-group fn-btn-group-xs">...</div>
</div>
</div>

垂直排列

使用 .fn-btn-group-stacked 使按钮垂直排列显示。 

						
<div class="fn-btn-group-stacked">
  <button type="button" class="fn-btn fn-btn-default">Button</button>
  ...
</div>

自适应宽度

添加 .fn-btn-group-justify class 让按钮组里的按钮平均分布，填满容器宽度。

使用 flexbox 实现，只兼容 IE 10+ 及其他现代浏览器 。

左手 中间 右手 

<div class="fn-btn-group fn-btn-group-justify">
  <a class="fn-btn fn-btn-default" role="button">左手</a>
  ...
</div>
