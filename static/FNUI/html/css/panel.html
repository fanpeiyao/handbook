Panel

面板组件带有轮廓、常用来放置带标题和文字的内容块。

基本样式

默认的 .fn-panel 提供基本的阴影和边距，默认边框添加 .fn-panel-default ，内容放在 .fn-panel-bd 里面。

这是一个基本的面板组件。
						
<div class="fn-panel fn-panel-default">
    <div class="fn-panel-bd">这是一个基本的面板组件。</div>
</div>

带标题的面板

.fn-panel-hd 用来放置标题，建议使用 h1 - h6 并添加 .fn-panel-title class，更加语义化。

面板颜色

添加不同的一下类可以设置不同的颜色。

  • .fn-panel-primary
  • .fn-panel-secondary
  • .fn-panel-success
  • .fn-panel-warning
  • .fn-panel-danger

面板页脚

面板页脚 .fn-panel-footer ，用于放置次要信息。页脚不会继承 .fn-panel-primary - .fn-panel-danger 等颜色样式。

面板内容
面板页脚
						
<section class="fn-panel fn-panel-default">
  <main class="fn-panel-bd">
    面板内容
  </main>
  <footer class="fn-panel-footer">面板页脚</footer>
</section>
