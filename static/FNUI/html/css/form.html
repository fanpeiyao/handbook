Form 表单元素

<form> 元素样式。

基本使用

单选、复选框

checkbox 、 radio 类型的 <input> 与其他元素稍有区别：

  • 块级显示时在容器上添加 .fn-checkbox、.fn-radio class；
  • 行内显示时在容器上添加 .fn-checkbox-inline、.fn-radio-inline class。

文件选择域

需要配合js使用 

						
<div class="fn-form-group fn-form-file">
  <button type="button" class="fn-btn fn-btn-danger fn-btn-sm">
    <i class="fn-icon-cloud-upload"></i> 选择要上传的文件</button>
  <input id="doc-form-file" type="file" multiple>
</div>
<div id="file-list"></div>
<script>
  $(function() {
    $('#doc-form-file').on('change', function() {
      var fileNames = '';
      $.each(this.files, function() {
        fileNames += '<span class="fn-badge">' + this.name + '</span> ';
      });
      $('#file-list').html(fileNames);
    });
  });
</script>

表单域状态

设置表单元素的不同状态。

禁用单个元素

给 <input> 添加 disabled 属性以禁用表单元素。 

						
<form class="fn-form">
  <input class="fn-form-field" type="text" placeholder="禁止输入的表单..." disabled>
</form>

给 <fieldset> 元素增加 disabled 属性，禁用所有的子元素。

禁用域内的元素 

						
<form class="fn-form">
  <fieldset disabled>
    <div class="fn-form-group">
      <label for="doc-ds-ipt-1">禁用的文本框</label>
      <input type="text" id="doc-ds-ipt-1" class="fn-form-field" placeholder="禁止输入">
    </div>

    <div class="fn-form-group">
      <label for="oc-ds-select-1">禁用的下拉选框</label>
      <select id="oc-ds-select-1" class="fn-form-field">
        <option>禁止选择我</option>
      </select>
    </div>

    <div class="fn-checkbox">
      <label>
        <input type="checkbox"> 无法选中的复选框
      </label>
    </div>

    <button type="submit" class="fn-btn fn-btn-primary">Submit</button>
  </fieldset>
</form>

禁用链接

<a> 元素设置禁用状态需要加上 .fn-disabled class。

禁止提交 
						
<a class="fn-btn fn-btn-primary fn-disabled">禁止提交</a>

表单排列

水平排列

在 <form> 添加 .fn-form-horizontal class 并结合网格系统使用。 

						
<form class="fn-form fn-form-horizontal">
  <div class="fn-form-group">
    <label for="doc-ipt-3" class="fn-u-sm-2 fn-form-label">电子邮件</label>
    <div class="fn-u-sm-10">
      <input type="email" id="doc-ipt-3" placeholder="输入你的电子邮件">
    </div>
  </div>

  <div class="fn-form-group">
    <label for="doc-ipt-pwd-2" class="fn-u-sm-2 fn-form-label">密码</label>
    <div class="fn-u-sm-10">
      <input type="password" id="doc-ipt-pwd-2" placeholder="设置一个密码吧">
    </div>
  </div>

  <div class="fn-form-group">
    <div class="fn-u-sm-offset-2 fn-u-sm-10">

行内排列

在外围容器上添加 .fn-form-inline 。 注意： 行内排列的元素并没有设置右边距，默认使用 inline-block 元素的间距，压缩 HTML 后行内表单元素的右边距会消失，需要自行处理。

 
 
  
						
<form class="fn-form-inline" role="form">
  <div class="fn-form-group">
    <input type="email" class="fn-form-field" placeholder="电子邮件">
  </div>

  <div class="fn-form-group">
    <input type="password" class="fn-form-field" placeholder="密码">
  </div>

  <div class="fn-checkbox">
    <label>
      <input type="checkbox"> 记住我
    </label>
  </div>

  <button type="submit" class="fn-btn fn-btn-default">登录</button>
</form>

表单域大小

单个域的大小

在表单元素上添加以下类名：

  • fn-input-lg
  • fn-input-sm

适用于没有 <label> 的表单，如果表单包含 <label> 且需要调整大小的，请往后看。 




						
<form class="fn-form">
  <input class="fn-form-field fn-input-lg" type="text" placeholder="添加了 .fn-input-lg">
  <br/>
  <input class="fn-form-field" type="text" placeholder="默认的 input">
  <br/>
  <input class="fn-form-field fn-input-sm" type="text" placeholder="添加了 .fn-input-sm">
  <br/>

  <div class="fn-form-group fn-form-select">
    <select class=" fn-input-lg">
      <option value="">添加了 .fn-input-lg</option>
    </select>
  </div>

  <div class="fn-form-group fn-form-select">
    <select class="">
      <option value="">select 默认大小</option>
    </select>
  </div>

  <div class="fn-form-group fn-form-select">
    <select class=" fn-input-sm">
      <option value="">添加了 .fn-input-sm</option>
    </select>
  </div>
</form>

组大小

在 .fn-form-group 的基础上添加以下 class，也可以实现对表单大小的设置：

  • .fn-form-group-sm
  • .fn-form-group-lg

注意可输入类型的 input 元素上需要添加 .fn-form-field ，不需要再添加 .fn-input-sm 此类的 class。 

						
<form class="fn-form fn-form-horizontal">
  <!-- fn-form-group 的基础上添加了 fn-form-group-sm -->
  <div class="fn-form-group fn-form-group-sm">
    <label for="doc-ipt-3-1" class="fn-u-sm-2 fn-form-label">电邮</label>
    <div class="fn-u-sm-10">
      <input type="email" id="doc-ipt-3-1" class="fn-form-field" placeholder="输入你的电子邮件">
    </div>

  <!-- fn-form-group 的基础上添加了 fn-form-group-lg -->
