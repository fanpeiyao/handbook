Button

定义按钮button元素

基本使用

在元素上添加.fn-btn，元素即成为按钮，再通过颜色设置相应的颜色。

  • .fn-btn-default - 默认，灰色按钮
  • .fn-btn-primary - 蓝色按钮
  • .fn-btn-secondary - 浅蓝色按钮
  • .fn-btn-success - 绿色按钮
  • .fn-btn-warning - 橙色按钮
  • .fn-btn-danger - 红色按钮
  • .fn-btn-link - 应用按钮样式的链接
链接 应用按钮样式的链接

圆角按钮

在默认样式的基础上添加 .fn-radiusclass. 

<button type="button" class="fn-btn fn-btn-default fn-radius">圆角样式</button>

椭圆形按钮

在默认样式的基础上添加.fn-roundclass. 


<button type="button" class="fn-btn fn-btn-danger fn-round">椭圆形按钮</button>

带边框按钮

在默认样式的基础上添加.fn-border-default


<button type="button" class="fn-btn fn-btn-default-border">默认样式</button>
<button type="button" class="fn-btn fn-btn-primary-border">主色按钮</button>
<button type="button" class="fn-btn fn-btn-secondary-border">次色按钮</button>
<button type="button" class="fn-btn fn-btn-success-border">绿色按钮</button>
<button type="button" class="fn-btn fn-btn-warning-border">橙色按钮</button>
<button type="button" class="fn-btn fn-btn-danger-border">红色按钮</button>

按钮状态

激活状态

在按钮上添加.fn-activeclass。



链接按钮激活状态 链接按钮激活状态 

<button type="button" class="fn-btn fn-btn-primary fn-active">激活状态</button>

禁用状态

在按钮上设置 disabled 属性或者添加 .fn-disabled class。



链接按钮禁用状态 链接按钮禁用状态 
<button type="button" class="fn-btn fn-btn-primary" disabled="disabled">禁用状态</button>

按钮尺寸

  • .fn-btn-xl
  • .fn-btn-lg
  • .fn-btn-default
  • .fn-btn-sm

<button class="fn-btn fn-btn-default fn-btn-xl">按钮 - xl</button>
<button class="fn-btn fn-btn-default fn-btn-lg">按钮 - lg</button>

块级显示

添加 .fn-btn-block class。 


<button type="button" class="fn-btn fn-btn-primary fn-btn-block">块级显示的按钮</button>
<button type="button" class="fn-btn fn-btn-danger-border fn-btn-block">块级显示的按钮</button>
