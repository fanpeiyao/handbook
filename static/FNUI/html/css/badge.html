Badge

Free Secondary Success Warning Danger

大小

结合辅助类中的字号 class，改变徽章大小。

default small normal large x large 

<span class="fn-badge fn-badge-primary">default</span> 
<span class="fn-badge fn-badge-secondary fn-text-sm">small</span> 
<span class="fn-badge fn-badge-success fn-text-default">normal</span>
<span class="fn-badge fn-badge-warning fn-text-lg">large</span>
<span class="fn-badge fn-badge-danger fn-text-xl">x large</span>
