基本边框

添加 .fn-table-bordered 类。 

						
<table class="fn-table fn-table-bordered">
    ...
</table>

圆角边框

同时添加 .fn-table-bordered 、 .fn-table-radius ，外层圆角边框通过 box-shadow 实现。 

						
<table class="fn-table fn-table-bordered fn-table-radius fn-table-striped">
    ...
</table>

单元格状态

表示表格状态的 class 添加到 tr 整行整行，添加到 td 高亮单元格。

  • .fn-active 激活；
  • .fn-disabled 禁用；
  • .fn-primary 蓝色高亮；
  • .fn-success 绿色高亮；
  • .fn-warning 橙色高亮；
  • .fn-danger 红色高亮。
Class 状态描述 目标元素
.fn-active 激活 td

  • .fn-table-striped 斑马纹效果
  • .fn-table-hover hover 状态

响应式表格

  • .fn-text-nowrap: 禁止文字换行；
  • .fn-scrollable-horizontal: 内容超出容器宽度时显示水平滚动条。

以上两个 class 在「辅助类」中定义。
