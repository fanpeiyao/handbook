Image

定义图片样式。

基础样式

基础样式定义在 base 中。

增强样式

圆角样式

为 <img> 元素设置不同的 class，增强其样式。

  • .fn-radius 圆角
  • .fn-round 椭圆
  • .fn-circle 圆形，一般用于正方形的图片(你要觉得椭圆好看，用在长方形上也可以)


<img class="fn-radius" alt="140*140" src="img/demos/02.jpg"width="140" height="140" />
<img class="fn-round" alt="140*140" src="img/demos/03.jpg" width="200" height="120" />
<img class="fn-circle" src="img/demos/01.jpg" width="140" height="140" />

边框

  • .fn-img-thumbnail 边框

<img class="fn-img-thumbnail" src="img/demos/01.jpg"/>
<img class="fn-img-thumbnail fn-radius" src="img/demos/02.jpg"/>
